Port forward on android phone

Baolong Lin

2018-01-25 15:13

Comments

Source

Phone reachable by hand most of time, sometime I use phone to do some work (ex, check internal email, code review comments, CI result, etc.). I need to login to one jump server (HOST1), then login another working server (HOST2). I'm using Termux on android, it's quit easy to access SSH tool. It work fine with tmux, and I run emacs inside it. When I need to access the web (gerrit, jenkins, etc), I need to do port forward. It can be achieve by SSH command, refer to an-ssh-tunnel-via-multiple-hops. On android phone, I don't want to type so many command or create one script to do that. I choose another SSH app ConnectBot. But need to do following step once.

  1. Create Host to HOST1 
    username@HOST1:22
  2. Add local port forward for HOST1 
    Type: Local
Source port: 9999
Destination: HOST2:22
  3. Create Host to HOST2 
    username@127.0.0.1:9999
  4. Add dynamic port forward for HOST2 
    Type: Dynamic(SOCKS)
Source port: 9998
  5. Configure the Firefox proxy

Open about:config change the following parameters: 

network.proxy.proxy_over_tls: true
network.proxy.socks: 127.0.0.1
network.proxy.socks_port: 9998
network.proxy.socks_remote_dns: true
network.proxy.socks_version: 5
network.proxy.type: 1

Now login HOST1, HOST2 with ConnectBot, use Firefox visit the internal website with the same way.
